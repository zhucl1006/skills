## 使用说明

1. **创建计划**: 复制模板到 `plans/{序号}-{功能名}.md`
2. **序号获取**: 查看 plans/ 目录，使用最大序号 +1
3. **状态流转**: 待执行 → 进行中 → 已完成/已暂停
4. **任务标记**: 完成时将 `[ ]` 改为 `[x]`
5. **执行记录**: 每完成一个任务添加一行记录

## 命名规范

- 文件名: 小写字母 + 连字符，如 `001-user-authentication.md`
- 功能名: 简洁描述，2-4个英文单词
- 序号: 三位数字，如 001, 002, 010
